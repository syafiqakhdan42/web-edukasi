HTML

HTML adalah bahasa dasar untuk membangun struktur sebuah halaman web. Berikut adalah contoh kode dasar HTML:

<!DOCTYPE html>
<html>
<head>
<title>Contoh HTML</title>
</head>
<body>
<h1>Halo Dunia!</h1>
</body>
</html>

CSS

CSS digunakan untuk mengatur tampilan halaman web. Berikut contoh kode CSS:

body {
  background-color: #f4f4f9;
  color: #333;
}
h1 {
  color: #007bff;
}

JavaScript

JavaScript digunakan untuk membuat halaman web interaktif. Berikut adalah contoh penggunaan JavaScript:

<script>
alert('Selamat Belajar JavaScript!');
</script>
